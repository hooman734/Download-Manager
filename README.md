### Download Manager

Python GUI download manager using `Tkinter`
it uses `pySmartDL` as the underlying module 
import time

* `python 3`

dependencies:
    + `pySmartDL`
    * `tkinter`
    * `tkinter.ttk`
    * `pathlib`
    * `threading`
    * `subprocess`
    * `os`
